# GitDigital Ledger Core

Financial + compliance ledger. Append-only. Events in, truth out.

## Overview

A robust, append-only digital ledger system designed for financial transactions and compliance monitoring. Built with immutability, auditability, and regulatory compliance as first-class citizens.

## Features

- **Append-only Architecture**: Once written, never modified
- **Cryptographic Integrity**: Hash chains and Merkle trees for verification
- **Compliance Engine**: Rule-based validation for regulatory requirements
- **Full Audit Trail**: Complete history of all events
- **REST API**: Easy integration with existing systems
- **PostgreSQL Storage**: Reliable, ACID-compliant persistence
